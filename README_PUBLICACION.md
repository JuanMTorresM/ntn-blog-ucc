Publicación rápida en GitHub Pages

1. Descarga y descomprime esta carpeta.
2. Sube todo el contenido a la raíz de tu repositorio.
3. En GitHub, ve a Settings > Pages.
4. En Source, elige Deploy from a branch.
5. Selecciona main y /(root).
6. Guarda y espera unos minutos.
7. Abre el enlace público que genere GitHub Pages.

Archivos principales:
- index.html
- styles.css
- app.js
- site-data.js
- assets/
