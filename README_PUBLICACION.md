# NTN Blog Premium Final

Sitio estático listo para GitHub Pages.

## Archivos principales
- index.html
- styles.css
- app.js
- site-data.js
- assets/

## Publicación rápida
1. Sube el contenido de esta carpeta al repositorio.
2. Verifica que `index.html` quede en la raíz.
3. Activa GitHub Pages en `Settings > Pages`.
4. Selecciona `Deploy from a branch`.
5. Usa la rama `main` y la carpeta `/(root)`.
