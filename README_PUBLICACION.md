# NTN Research Atlas

## Publicación en GitHub Pages
1. Descarga y descomprime esta carpeta.
2. Sube el contenido completo del proyecto a la raíz de tu repositorio.
3. En GitHub ve a **Settings > Pages**.
4. En **Build and deployment** selecciona:
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** /(root)
5. Guarda los cambios y espera unos minutos.
6. Recarga tu enlace publicado.

## Estructura
- `index.html`: interfaz principal
- `styles.css`: estilos premium y responsive
- `app.js`: interacciones, pestañas, asistente y mapa
- `site-data.js`: contenido del sitio
- `assets/`: ilustraciones, gráficas y datos del mapa

## Nota técnica
El proyecto está listo para publicarse como sitio estático en GitHub Pages.
